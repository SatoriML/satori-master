# Generated Campaign Assets

```
linkedin/
  ├── company/    # Företagskonto-inlägg, planeringar, favoriter
  └── joachim/    # CEO-personligt innehåll
email/             # Outreach-sekvenser och e-postmallar
research/
  ├── analytics/  # KPI-sammanställningar och mätpunkter
  └── market/     # Bransch- och marknadsrapporter
calendar/
  └── plans/      # Planeringsunderlag och innehållskalendrar
seo/               # SEO-briefs och sökordsstrategier
```

- Namnge filer `YYYY-MM-DD_topic.md` när tid är kritisk, annars använd tydliga ämnesnamn.
- Flytta färdigpublicerade paket till `archive/<Quarter>/campaigns/` vid kvartalsslut.
- Håll referenser i `.claude/agents/*` uppdaterade om du flyttar en kanal.
