# Pipeline Workspace

```
stages/
  ├── Fas1/  # Nya leads
  ├── Fas2/  # Kvalificering & behovsanalys
  ├── Fas3/  # Slutförhandling
  └── Won/   # Vunna affärer tills de flyttas till clients/
opps/        # Snabbparkering för möjligheter utan struktur ännu
```

- Lägg prospektlistor, win/loss-analyser och forecast-filer här.
- För prospects utan kundmapp: skapa `stages/FasX/<Företag>/` och följ client-templatet (mötesanteckningar i `1.Möten/`).
- Flytta vunna affärer till `clients/<Företag>/` när kontrakt är klart.
- Arkivera stängda/förlorade kalkyler under `archive/<Kvartal>/`.
